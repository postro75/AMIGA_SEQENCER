# Amiga sample sources

Original Soundtracker / ProTracker **ST-xx** sample disks:

- Aminet: https://aminet.net/mods/inst (st-01.lha, st-02.lha, …)
- Internet Archive: https://archive.org/details/AmigaSTXX (Public Domain Mark 1.0)

Fetched with `python fetch_amiga_samples.py`.

Demo MOD (`MOD/axelf.mod` — "elekfunk !" by moby) from The Mod Archive for instrument extraction tests.

These packs are historical demoscene / tracker culture material redistributed for preservation.
